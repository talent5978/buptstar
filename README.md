# 工程思政数字教育资源库 (buptstar)

This project was reconstructed from build artifacts. It is a React application built with Vite and Tailwind CSS.

## Project Structure

- `src/`: Source code
  - `components/`: React components
  - `pages/`: Page components
  - `data/`: Data files extracted from the build
  - `assets/`: Images and other assets
  - `App.jsx`: Main application component
  - `main.jsx`: Entry point
- `public/`: Static assets (images)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Notes

- The project uses `react-router-dom` for routing with a base path of `/buptstar`.
- Tailwind CSS is configured for styling.
- Icons are provided by `lucide-react`.
